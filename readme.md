# Setup
## Clone the repo
```bash
git clone https://github.com/vishnubv944/harmoni-frontend.git
```

## Install the dependencies 
```bash
cd harmoni-frontend
```
```bash
npm install
```
```bash
npm i -g @ionic/cli
```

```bash
npm i @capacitor/core
npm i -D @capacitor/cli
```

## Open the project through android studio

```bash
ionic capacitor open android
```
